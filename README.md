## :clipboard: Tech Stack
- Minimum SDK level 21(Android 5.0)
- Programming language: Kotlin
- Clean Architecture + MVVM pattern
- Network: Retrofit + OkHttp
- DI: Hilt
- Async: Coroutines + LiveData
- DataBase: Room
